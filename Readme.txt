【概要】
通話録音からテキストに自動的に書き起こすソフトです。
インストール方法や使い方は以下の動画を見てください。2つとも内容は同じです。
https://www.nicovideo.jp/watch/sm39257003
https://youtu.be/pCebMJgtVBM

【アンインストール】
配置したファイル・フォルダを削除してください。

【著作権/再配布に関して】
本ソフトウェアはフリーウェアですが、著作権は放棄しておらず、「ろじろじ」が保有しています。
無断転載/再配布はご遠慮ください。
連絡はReadme先頭に記載してあるメールアドレス、もしくはTwitterにお願いします。

【商用利用について】
音声認識に使用しているAPIの利用規約に、
「You and any API Client you create may only access this API for your own development and personal uses.」
とあります。上記に反しない範囲でDisNOTEを使用してください。

【ソフトの改変について】
ソースコードがそのまま入っています。改変も制限しません。
複数の改良版が公開されて混乱することを防ぐため、改変したソフトウェアの無断での再配布は禁止します。
改変版の公開を考えている方は、作者（ろじろじ）にご連絡ください（開発を引き継いでいただくかもしれません）。
